# Tourist Spots — Static Site (Ready for GitHub Pages)

This mini static website showcases a few tourist spots and demonstrates deploying to GitHub Pages.

## Files
- index.html
- css/styles.css
- js/script.js
- images/* (place images here; the uploaded image is included as whatsapp-upload.jpeg)

## Run locally
Open `index.html` in a browser.

## Deploy
1. Push this repo to GitHub.
2. In repo Settings → Pages, select branch `main` and folder `/ (root)`.
3. Save — your site will be published at: `https://<your-github-username>.github.io/<repo-name>/`
